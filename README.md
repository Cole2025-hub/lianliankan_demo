# 连连看 - 经典消除游戏

## 简介

一款精美的 HTML5 连连看消除游戏，无需安装，打开即可玩！游戏包含 10 个精心设计的关卡，支持提示、暂停、连击计分等功能，并带有流畅的动画效果和星空主题 UI。

## 快速开始

下载 lianliankan_game.html文件，双击在浏览器中打开即可开始游戏。

## 游戏特色

- **10 个关卡**：难度逐渐递增，从 4x6 到 6x10 的棋盘，图标种类从 6 种增加到 15 种
- **连线轨迹**：成功消除时显示金色连线动画，清晰展示连接路径
- **提示功能**：每局 3 次提示机会，高亮显示可消除的图标对及连接路径
- **暂停/继续**：随时暂停游戏
- **连击系统**：连续成功消除获得额外分数加成
- **时间奖励**：每关通关后，剩余时间将转换为额外分数
- **精美 UI**：星空背景、渐变色彩、流畅动画效果
- **跨平台**：支持所有现代浏览器（Chrome、Firefox、Edge、Safari）

## 玩法规则

1. 点击两个相同的图标进行消除
2. 两个图标之间必须可以通过一条不超过 2 个转弯的路径连接
3. 路径可以经过空白区域（已消除的位置）或棋盘外围
4. 清空棋盘即可过关
5. 每关有倒计时，时间用尽则游戏失败

## 计分规则

- 每次消除基础分：100 分
- 连击加成：第 N 次连击额外获得 N x 10 分
- 通关时间奖励：剩余秒数 x 10 分

## 技术栈

- HTML5 / CSS3 / JavaScript（纯原生，无第三方依赖）
- SVG 用于绘制连线轨迹动画
- CSS3 动画实现各种交互效果

## 开发说明

### 修改关卡配置

编辑 `lianliankan.html` 中的 `LEVELS` 数组：

```javascript
const LEVELS = [
    { rows: 4, cols: 6, pairs: 12, time: 120, emojiCount: 6 },
    // rows: 行数, cols: 列数, time: 限时(秒), emojiCount: 图标种类数
];
```

### 更换图标

编辑 `EMOJIS` 数组，替换为你想要的 emoji 图标：

```javascript
const EMOJIS = ['🍎', '🍊', '🍋', '🍇', '🍓', '🍒', ...];
```
