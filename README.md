# 水果连连看

一个使用 Python Tkinter 编写的桌面版连连看小游戏。新版不再使用纯字母按钮，而是用彩色卡片、emoji 水果图案、选中高亮、提示高亮和连线路径来呈现，更像一个真正的小游戏界面。

## 功能特点

- Canvas 绘制的彩色水果卡片界面，不依赖系统 emoji 字体
- 3 个关卡，棋盘从 6x6 逐步提升到 10x10
- 支持经典连连看规则：直线、一次转弯、两次转弯
- 点击可连接的相同图案后会显示消除连线
- 提供提示功能，会高亮一对可消除图案
- 提供洗牌功能，遇到无解棋盘时可重新排列剩余图案
- 支持重开当前关卡
- 无第三方依赖，直接使用 Python 标准库 Tkinter

## 运行环境

- Python 3.x
- Tkinter

大多数 Windows 和 macOS 的 Python 安装包会自带 Tkinter。Linux 用户如果缺少 Tkinter，可以根据发行版安装对应包，例如：

```bash
sudo apt install python3-tk
```

## 运行方法

在项目目录下执行：

```bash
python lianliankan_game.py
```

## 游戏规则

1. 点击一个水果卡片进行选择。
2. 再点击另一个相同图案的卡片。
3. 如果两个图案之间可以通过空白路径连接，并且路径最多只转弯两次，就会消除这一对。
4. 清空当前棋盘后进入下一关。
5. 完成三关后游戏会重新开始。
